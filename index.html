GIT INIT

git init
Git 저장소 초기화
git status
Git 상태(새롭게 추가된 내용이나 수정된 내용) 확인
dit add [*, filename]
파일 버전관리 스냅샷 생성
git commit -m "메모 내용"
영구적으로 버전관리 스냅샷 파일 기록
git commit -a -m "메모 내용"
스테이지 상태를 건너뛰고 커밋
git diff
스테이지(stage)되지 않은 변경사항을 화면에 출력
git diff --staged
스테이지된 이후 변경사항을 화면에 출력
git log
커밋(commit)된 결과를 화면에 출력
